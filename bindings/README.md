# Forwarder Bases Bindings

This package provides [Rust](https://www.rust-lang.org/) bindings to the Solidity contracts.

## Prerequisites

1. Get an up-to-date version of [Rust](https://www.rust-lang.org/) with

   ```sh
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

## Usage

### Build

Run

```sh
cargo build
```

### Test

To test the build, run

```sh
cargo test
```
